# Coin Flip Game - 玩法指南

## 游戏概述

Coin Flip Game 是一个简单的硬币翻转猜测游戏。玩家猜测硬币正面或反面, 猜对赢取金币, 猜错扣除金币。

## 基本规则

1. 初始金币: 100 枚
2. 每次下注: 10 枚金币
3. 猜对: 获得 +10 金币 (净赚 10 枚)
4. 猜错: 失去 10 金币
5. 金币不足 10 枚时游戏结束

## 操作流程

1. 点击 "Heads" 或 "Tails" 选择猜测
2. 点击 "Flip Coin" 翻转硬币
3. 等待动画结束, 查看结果
4. 结果显示为绿色 (赢) 或红色 (输)

## 界面元素

- **Your Coins**: 显示当前金币数量
- **硬币图标**: 显示翻转动画
- **Heads/Tails 按钮**: 选择猜测
- **Flip Coin 按钮**: 执行翻转 (需先选择猜测)
- **结果区域**: 显示本轮结果
- **History**: 显示历史记录

## 预期行为

- 选择猜测后, Flip Coin 按钮变为可用
- 翻转过程中按钮不可点击 (防止重复操作)
- 金币变化应与结果一致 (赢 +10, 输 -10)
- 历史记录按时间倒序排列
